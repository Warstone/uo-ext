================================================================================
= Ultima Online client and protocol extender server side integration for RunUO =
= Интеграция RunUO серверной части Ultima Online client and protocol extender  =
================================================================================

Установка:
	
Вариант 1 (для него требуется чтобы ваш сервер работал на .NET 4.0, 
		   для этого может потребоваться пересобрать ваш сервер)	
	1) Скомпилировать конфигурацию "Release" проекта "UOExtDomain (dll).csproj", 
	полученную сборку "UOExtDomain.dll" поместить в корневую папку сервера (в 
	директорию, где располагается "Server.exe")
	2) [Опционально] Поместить в корневую папку следующие библиотеки из "./../":  
	"bzip2(x64).dll", "bzip2(x86).dll", "zlib(x64).dll", "zlib(x86).dll"
	3) Добавить ссылку на сборку "UOExtDomain.dll" в проект "Scripts"
	4) Включить в проект "Scripts" элементы из папки "./Scripts"
	5) Отредактировать настройки сервера в ".\Scripts\Misc\ClientExtender.cs"
	(подробнее о настройках читайте в коментариях в коде)
	6) определите при компиляции скриптов одну из констант NET_4_0, NETCF_4_0, MONO_4_0
	7) Если сервер не использует авто-компиляцию, перекомпилируйте проект "Scripts".
