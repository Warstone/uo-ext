================================================================================
= Ultima Online client and protocol extender server side application           =
= Приложение серверной части Ultima Online client and protocol extender 	   =
================================================================================


Установка:
	
	1) Скомпилировать конфигурацию "Release" проекта "UOExtDomain (exe).csproj", 
	полученную сборку "UOExtDomain.exe" поместить в желаемую папку.
	2) [Опционально] Поместить в корневую папку следующие библиотеки из "./../":  
	"bzip2(x64).dll", "bzip2(x86).dll", "zlib(x64).dll", "zlib(x86).dll"
	3) Создайте папку "UOExtPlugins" в корневой папке сервера и поместите туда 
	желаемые плагины, с именем в формате "Plugin-0x??.dll"
	4) Поместите в корневую папку файл "./Config.ini" и отредактируйте его.
